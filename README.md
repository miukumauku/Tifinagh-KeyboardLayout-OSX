Tifinagh keyboad layout for MacOS X
===================================

# Install
1. Download "Tifinagh.keylayout".
2. Move "Tifinagh.keylayout" into ~/Library/Keyboard\ Layouts/.
3. Add Input Method "Tifinagh (Y.U.U,)" on Preferences.

# Layout
[Normal]
   
      ` 1 2 3 4 5 6 7 8 9 0 - =
          ⵇ ⵡ ⴻ ⵔ ⵜ ⵢ ⵓ ⵉ ⵖ ⵒ [ ] \
           ⴰ ⵙ ⴷ ⴼ ⴳ ⵀ ⵊ ⴽ ⵍ ; '
            ⵣ ⵅ ⵛ ⵠ ⴱ ⵏ ⵎ , . /

[Shift]
   
      ~ ! @ # $ % ^ & * ( ) _ +
         ⵈ ⵯ ⵦ ⵕ ⵟ ⵢ ⵓ ⵉ ⵄ ⵒ { } |
           ⴰ ⵚ ⴹ ⴼ ⴶ ⵃ ⵌ ⴾ ⵍ : "
            ⵥ ⵆ ⵞ ⵠ ⴱ ⵐ ⵎ < > ?

[Option]
   
                   ⵝ       ⵧ
                ⴸ   ⴴ ⵁ ⵋ ⴿ
             ⵤ       ⴲ  ⵑ

[Option Shift]
   
                            ⵘ
              ⴺ   ⴵ ⵂ
